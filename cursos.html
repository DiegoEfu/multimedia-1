Cursos de Machine Learning

Machine Learning con Python

Ofrecido por: IBM.

  Este es un curso introductorio de Python y de Machine Learning en general Ofrecido por la prestigiosa empresa IBM en la plataforma de Coursera. El plan de estudio incluye temas básicos del Machine Learning como la regresión, la clasificación y el clustering.

Especialización de Machine Learning

Ofrecido por: Stanford University y DeepLearning.AI.

  Esta especialización de 3 cursos sobre fundamentos de Machine Learning fue creada en una colaboración entre DeepLearning.AI y Stanford Online. Este curso es perfecto para principantes, y enseñará como utilizar algoritmos de machine learning para crear aplicaciones del mundo real.
Los cursos de la especialización son:


  • Aprendizaje Automatizado Supervisado: Regresión y Clasificación.
  • Algoritmos Avanzados de Aprendizaje.
  • Aprendizaje no Supervisado, Recomendadores y Aprendizaje por Refuerzo.

Certificación de TensorFlow Developer

Ofrecido por: DeepLearning.AI.

  Este es un curso de certificación en TensorFlow que consta de 4 cursos, donde el estudiante aprenderá las herramientas necesarias para construir aplicaciones basadas en inteligencia artificial con TensorFlow. Al finalizar el programa, el estudiante recibirá el certificado de desarrollador de TensorFlow de Google.
Los cursos de la especialización son:


  • Introducción a TensorFlow para Inteligencia Artificial, Machine Learning y Deep Learning.
  • Redes Convolucionales en Tensorflow.
  • Procesamiento de Lenguaje Natural con TensorFlow.
  • Secuencias, Series Temporales y Predicciones.
