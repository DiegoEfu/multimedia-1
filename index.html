¿Qué es el Machine Learning?

  El "machine learning", o aprendizaje automático, es una rama de la inteligencia artifical (IA) que permite que las máquinas aprendan sin ser expresamente programadas para ello. Es decir, es la habilidad que tienen las máquinas para hacer sistemas capaces de identificar patrones entre los datos para hacer predicciones a través de un modelo matemático o estadístico del problema y un algoritmo de aprendizaje adecuado.


Aprendizaje Automatizado, o Machine Learning

¿Cuál es su origen?

  En 1950 el matemático Alan Turing planteó por primera vez la pregunta de si es posible que las máquinas puedan pensar, con la que plantó la semilla de la creación de computadoras de "inteligencia artificial". De esa forma surgió la idea de computadoras capaces de replicar de forma autónoma tareas típicamente humanas, como la escritura o el reconocimiento de imágenes.


Alan Turing

  Fue un poco más adelante, entre las décadas de 1950 y 1960, cuando distintos científicos empezaron investigar cómo aplicar la biología de las redes neuronales del cerebro humano para tratar de crear las primeras máquinas inteligentes. Sin embargo, debido a la poca cantidad de datos disponibles, no fue sino hasta la actualidad que el Machine Learning empezó a ser viable gracias al aumento de la disponibilidad de datos y del desarrollo del poder computacional.

¿Para qué sirve?

  • Finanzas: Las tecnologías inteligentes pueden ayudar a los bancos a detectar el fraude, predecir patrones del mercado y aconsejar operaciones a sus clientes.
  • Asistentes Virtuales: Permite saber si un estudiante está a punto de cancelar su registro, sugerir nuevos cursos o crear ofertas personalizadas para optimizar el aprendizaje.
  • Comercio: Posibilita hacer pronósticos de ventas y elegir el producto adecuado para recomendárselo al cliente.
  • Logística y Transporte: El ML es útil a la hora de predecir colisiones o atascos y también para optimizar el tráfico.
